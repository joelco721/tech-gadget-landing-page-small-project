# Tech Gadget Landing Page

A small responsive landing page project built to practice my HTML and CSS skills.

This project was created as part of my frontend learning process. The main focus was on page structure, layout, styling, responsive design, and basic user interface design. I also added a small JavaScript interaction for button click alerts.

## Technologies Used

- HTML
- CSS
- Basic JavaScript

## What I Practiced

- Building a webpage structure with HTML
- Styling sections, buttons, cards, and navigation with CSS
- Using Flexbox for layout
- Creating hover effects
- Making the page responsive with media queries
- Adding a basic JavaScript click event

## Project Status

This is a beginner frontend practice project.  
The main goal was to improve my HTML and CSS skills, with only a small amount of JavaScript added.
