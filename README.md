# Tech Gadget Landing Page

A small responsive landing page project built completely by me from scratch.

I designed and built this project without using AI, without following a tutorial, and without copying a template. The goal was to test and practice my own HTML and CSS skills and see what I could create independently.

The main focus of this project was page structure, layout, styling, responsive design, and basic user interface design. I also added a small JavaScript interaction for button click alerts.

## Technologies Used

- HTML
- CSS
- Basic JavaScript

## What I Practiced

- Creating a full landing page from scratch
- Structuring a webpage with HTML
- Designing the layout and visual style independently
- Styling navigation, hero section, buttons, and content cards
- Using Flexbox for layout
- Creating hover effects
- Making the page responsive with media queries
- Adding a basic JavaScript click event

## Project Status

This is a beginner frontend practice project.

The main goal was to prove to myself that I can design and build a landing page independently using my current HTML and CSS knowledge.
